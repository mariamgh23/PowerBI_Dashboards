## 📌 How to Use

Each project lives in its own folder. Inside you'll find:
- A `README.md` with context, visuals, and key findings
- The `.pbix` source file (open with [Power BI Desktop](https://powerbi.microsoft.com/desktop/))
- A `data/` folder with the raw dataset used
- An `assets/` folder with a dashboard screenshot

---

## 🚀 Getting Started

1. Clone the repo
   ```bash
   git clone https://github.com/YOUR_USERNAME/data-visuals.git
   ```
2. Navigate to any project folder
3. Open the `.pbix` file with Power BI Desktop
